WHAT IS EACH FILE?
==================

app.py
  └─ Main Streamlit dashboard
     • Live camera feed with detection
     • Real-time metrics
     • Violation logs
     • Email alerts
     • Analytics charts

config.py
  └─ Centralized configuration
     • Loads .env variables
     • Sets default values
     • Creates required directories

safetyeye/logger.py
  └─ Application logging
     • File and console output
     • Daily log rotation
     • Configurable levels

safetyeye/detector.py
  └─ YOLO detection engine
     • Model loading
     • Frame detection
     • Violation identification
     • Box drawing

safetyeye/alerts.py
  └─ Email alert system
     • Gmail SMTP
     • Cooldown mechanism
     • Error handling

safetyeye/database.py
  └─ Violation logging
     • CSV management
     • Data queries
     • Statistics
